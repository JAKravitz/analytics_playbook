## Spectral uplift
- **MSI baseline:** Gradual trends in greenness where signal allows.
- **VNIR HSI adds:** Canopy biochemical decline (cab, cw) detectable before structural thinning visible in MSI.
- **VNIR-SWIR HSI adds:** SWIR-derived structural thinning (cm, lignin) detectable earlier and more precisely — key for selective logging and fragmentation detection.
